I develop and maintain [Pipocas.tv](https://pipocas.tv), a Portuguese subtitles community, and contribute as a moderator at [TheTVDB](https://thetvdb.com).

## ✨ Features

- 🌑 **Dark Theme** — Catppuccin Mocha palette
- 🇵🇹 / 🇬🇧 **Dual Language** — pt-PT / EN with dynamic language switching
- 📱 **Fully Responsive**
- ⚡ **Zero Build Step** — simple static site, easy to maintain
- ♿ **Accessible** — skip-link, focus states, `prefers-reduced-motion`
- 🎬 **Smooth Animations** — staggered section reveals
- 🎵 **Spotify Embed** — with graceful fallback
- 🧩 **Projects Section** — clean interactive cards

## ▶️ Run Locally

No setup is required.

```bash
git clone https://github.com/Blackspirits/blackspirits.github.io.git
cd blackspirits.github.io
```

Then open `index.html` in your browser.
